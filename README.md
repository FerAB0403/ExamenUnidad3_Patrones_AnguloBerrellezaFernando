Nombre del proyecto asignado: Gestor de reportes con distintos formatos de salida

Descripción: Este programa en C# simula la decoración  de un reporte (decorarlo con gráficos ya se de barra o de pastel, ademas de cambiarle la fuente y su color) credo por consola utilizando el patrón de diseño decorador (Decorator), además, de "poder exportar" el reporte en formato de documento (PDF, Excel, Word, PowerPoint) o imagen (JPG, PNG) utilizando el patrón compuesto (Composite).

Abrir la carpeta "src".
Despues descargar la carpeta llamada "U3Examen".
Ya descargada la carpeta, se debe de abrir y dar doble clic en la solución (U3Examen.sln).
Una vez inicializado el programa seleccionar la opción "Iniciar" para ejecutar el código.
